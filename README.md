# BulkyBook

## Overview

BulkyBook is a Category Management web application built using ASP.NET Core MVC and Entity Framework Core.

The project demonstrates the implementation of CRUD operations, database integration, cloud deployment, and modern web development practices using Microsoft's technology stack.

---

## Features

* Create Categories
* View Categories
* Update Categories
* Delete Categories
* Server-side Validation
* Entity Framework Core Integration
* SQL Server Database
* Azure Cloud Deployment

---

## Technology Stack

### Backend

* ASP.NET Core MVC
* Entity Framework Core

### Database

* SQL Server
* Azure SQL Database

### Frontend

* Razor Views
* Bootstrap
* jQuery

### Cloud

* Azure App Service
* Azure SQL Database

## Getting Started

### Prerequisites

* Visual Studio 2022
* .NET 8 SDK
* SQL Server

### Installation

1. Clone the repository
2. Configure appsettings.json
3. Apply Entity Framework migrations
4. Run the application
